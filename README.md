## 🎮 RPG Game System (New!)
Alice kini dilengkapi dengan sistem RPG yang mendalam dan profesional:
- **Core Gameplay**: `.adventure`, `.hunt`, `.mining`, `.fishing`, `.work`, `.dungeon`, `.battle`, `.quest`
- **Character**: `.register`, `.profile`, `.inventory`, `.stats`, `.skill`, `.levelup`
- **Economy**: `.shop`, `.sell`, `.use`, `.craft`, `.upgrade`, `.transfer`, `.market`
- **Competitive**: `.leaderboard`, `.arena`, `.guild`, `.tournament`

## 👥 Group Management
- **Auto Features**: Welcome/Goodbye messages, Anti-link, Anti-spam, Auto-kick bad words.
- **Moderation**: `.kick`, `.add`, `.promote`, `.demote`, `.mute`, `.warn`.
- **Info**: `.groupinfo`, `.tagall`, `.listonline`.

## 🤖 AI Features
- **ChatGPT & Gemini**: Integrasi AI cerdas untuk tanya jawab.
- **AI Image Generation**: Buat gambar dari teks dengan `.imagine` atau `.flux`.

## 🛠️ Utilities & Tools
- **Media**: `.sticker`, `.toimg`, `.play`, `.ytmp3`, `.ytmp4`, `.tiktok`.
- **Tools**: `.tts`, `.translate`, `.ocr`, `.weather`, `.wiki`.

## 🔧 Installation

1. **Clone Repository**
   ```bash
   git clone https://github.com/hiuraaaaa/Alice-Bot.git
   cd Alice-Bot
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Configure Settings**
   Edit file `config/bot.config.js` dan `config/rpg.config.js`.

4. **Run Bot**
   ```bash
   npm start
   ```
